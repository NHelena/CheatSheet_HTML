HTML cheat-sheet

You'll find below a list of the HTML most-commonly used tags and their respective short description.
Feel free to share your thoughts by hitting the "Contact" button up above.

  • Structure
  • Link
  • List

Structure tags
<!doctype html> Set the page type to HTML
<html>[...]</html> Contains all the HTML content
<head>[...]</head> Contains all the navigator meta-data
<title>[...]</title> Set a specific title to the web page
<h[1-6]>[...]</h[1-6]> Heading (h1 being the largest & h6 the smallest)
<body>[...]</body> Contains all the content of the page
<header>[...]</header> Contains the body header
<footer>[...]</footer> Contains the body footer
Linking tags
<img src="[url]"> Display an image
<a href="[url]"> Basic link to a website
<a href="#[id]"> Anchor link
<a href="[url]#"> Anchor link to another website
<a href="mailto:[email]"> Send an email to a given address
<a href="mailto:[email]?Subject=[subject]"> Send an email to a given address with a specific subject
List tags
<ol>[...]</ol> Ordered list
<ul>[...]</ul> Un-ordered list
<li>[...]</li> List item (within [un-]ordered list)
<dl>[...]</dl> Definition list
<dt>[...]</dt> Term or phrase being defined
<dd>[...]</dd> Detailed definition of a term or a phrase
Miscellaneous tags
<br> Line break
<hr> Horizontal line
